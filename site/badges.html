Why self-hosted

We don't use shields.io. The badges live at asclepius.health/assets/badges/*.svg and are part of the same asset-manifest.json SHA-256 record that every other page is. If a badge ever shows you a value we cannot back up with a published bundle, the manifest will say so.

Hosts that proxy SVG (e.g., GitHub's image proxy camo.githubusercontent.com) will cache the badge for several minutes. That's fine — the values change rarely, and the page that authoritatively reports each value (/changelog, /status, /conformance) is one click away.

For partners + design partners

If you're piloting Asclepius and want a "powered by" pill for your hospital's intranet, the same set fits. Email labs@asclepius.health if you'd like a customized monochrome variant — we keep them in assets/badges/mono/ when partners need to drop them onto a coloured background.
