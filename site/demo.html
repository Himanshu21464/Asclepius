· disclaimer · This page is a teaching surface. The C++ runtime uses BLAKE2b-256 and Ed25519 against a SQLite-backed Merkle ledger; the demo uses SHA-256 and (where supported) Ed25519 against an in-memory array. Signatures generated here verify with the same public key in your tab, not the lab next door — a real bundle compares against the customer's published ledger head.
